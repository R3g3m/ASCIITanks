# ASCIITanks
Classic game 

Классические танки на приставку Sega, только отрисовка графики выполняется в консоли с помощью ASCII кодов.
Реализован проект будет на С++, без всяких дополнительных движков и графических библиотек, возможно с использованием ассемблера для ускорения отрисовки и упрощения работы с видеопамятью.
